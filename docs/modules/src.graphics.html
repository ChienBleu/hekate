Functions

get_tileset_quad (tile:, x:, y:, h:, v:, d:, seed:)
Return a tileset quad given the tile and its neighbour.

Parameters:

  • tile: The number representing the kind of tile to evaluate.
  • x: The x position of the tile.
  • y: The y position of the tile.
  • h: Is there an horizontal neighbour ?
  • v: Is there a vertical neighbour ?
  • d: Is there a diagonal neighbour ?
  • seed: The seed to randomize the tile.
pixel_to_tile (x:, y:, scale:)

Parameters:

  • y: The y in pixel.
  • scale: The scale of the camera.
