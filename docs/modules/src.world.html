Module src.world

A module that is used to keep track of the world's components.

Functions

select (filter:) Return all the components that match the filter.
all () Return all the components in the world.
add (component:) Add a new component to the world.
delete (entity:, component:) Delete a component.
get (entity:, component:) Return the component specified by its name and its entity.
get_by_key (key:) Return the component specified by its key.
exists (entity:, component:) Return true if the component specified by its name and its entity exists.
load (world_name:) Use to set the world to a specific folder.
serialize (cluster:) Serialize all the components into the world file.
